WhatsApp Chat Summarizer

Overview:
A simple tool that summarizes exported WhatsApp chat files (.txt) based on user-defined number of recent messages.

Features:
• Upload WhatsApp exported chat file
• Select number of recent messages to summarize
• Generates concise summary of conversation

How it works:
1. User exports chat from WhatsApp
2. Uploads .txt file
3. System processes text and extracts recent messages
4. Generates summary output

Tech Used:
• Python (or whatever you used)
• Text processing / basic NLP techniques

Future Improvements:
• Add sentiment analysis
• Support for group chat insights
• Web interface instead of manual upload
